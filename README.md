# Etch a Sketch 

This project consists of creating a web version of Etch a Sketch by placing a grid of divs or squares on the screen.
As the user moves the mouse over the grid, each individual square gets assigned a random rgb value.

With every mouse interaction, the square will increase its opacity. After 10 interactions it reaches full opacity.

The user can also click a button on top of the screen to modify the grid.
It sends a pop-up asking for a positive integer value (less than 100), and it creates a new grid with the given dimensions.
